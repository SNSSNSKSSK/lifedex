# 🌱 LifeDex

> 결과보다 경험을 수집한다.

LifeDex는 삶 속에서 경험한 모든 것을 기록하고,
시간이 지날수록 나만의 인생 도감을 만들어가는 웹 서비스입니다.

---

## ✨ Vision

사람들은 목표보다 수많은 경험을 하며 살아갑니다.

하지만 대부분의 경험은 기록되지 않고 잊혀집니다.

LifeDex는 경험을 기록하고,
시간이 흐를수록 나만의 인생을 시각적으로 돌아볼 수 있는 서비스를 목표로 합니다.

---

## 🚀 MVP
